## Our Usage in This Project

### Scope
The World English Bible Updated Edition (WEB-UE) is used as the authoritative Scripture text source for:
- All psalm texts in section 8.0 (Psalm Settings)
- All canticle texts in section 6.0 (Canticles)
- All scriptural quotations in hymn texts
- All Divine Service liturgical texts with Scripture content

### Attribution
Each file using WEB-UE text includes:
```
Psalm: N (World English Bible Updated Edition)
```

or

```
Words: [Scripture reference] World English Bible Updated Edition; public domain.
```

### Compliance
This project:
- ✅ Does not modify the text (except for line breaks in poetry)
- ✅ Maintains the WEB-UE name attribution
- ✅ Acknowledges public domain status in all citations
- ✅ Allows free distribution and modification by users
- ✅ Respects the trademark in naming
